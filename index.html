Run on Colab (No Install)

Upload app.py and your song, run one command, download stems.

!pip -q install demucs librosa numpy soundfile from google.colab import files files.upload() !python app.py "song.mp3" --out outputs --format wav

Download Output

Zip the outputs and download to your device.

import shutil from google.colab import files shutil.make_archive("stems", "zip", "outputs") files.download("stems.zip")

CLI Usage

python app.py "song.mp3" --out outputs --format wav python app.py "song.mp3" --out outputs --format mp3 python app.py "song.mp3" --json

Notes

Only process audio you own or have rights to. Tempo/key are estimates and depend on input quality.
